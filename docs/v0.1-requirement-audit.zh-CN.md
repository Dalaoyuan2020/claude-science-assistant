## 7. v0.1.3 发布仍需注意

- GitHub 当前公开版本为 v0.1.2；本地 v0.1.3 候选尚未提交、推送或创建 Release。
- 309 或另一台不同环境电脑的升级/安装验收仍是发布门槛。
- 提交后必须重新编译并打包，不能使用 Release `-SkipBuild`；远端同时上传 ZIP 与同名 `.sha256`。
- 内置中转验证时返回 502，恢复前不要宣称其文本、工具或 reasoning 能力已通过。
- 任何 issue、截图、日志、证据包都不能包含真实 API Key。
